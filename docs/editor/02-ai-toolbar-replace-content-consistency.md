# AI 工具栏替换内容一致性修复

- Version: 1.0.0
- Last Updated: 2026-03-06
- Code Paths:
  - apps/frontend/src/components/GlobalAiToolbar.vue

## 功能目的
确保全局 AI 工具栏“替换选中文本”时，展示内容与 `api/ai/generate` 接口返回保持一致，不在前端再次二次清洗导致内容缺失。

## 使用方式/入口
- 入口：页面可编辑区域选中文本后点击全局 AI 工具栏（翻译/润色/解释等）动作。
- 关键行为：请求成功后，直接将接口返回文本用于替换选区内容。

## 关键约束与边界
- 思考内容清洗统一在后端进行，前端替换层不再重复过滤。
- 前端仅负责：保留选区、调用接口、执行替换与错误提示。

## Changelog
- 2026-03-06 **Fix**: 移除前端 `GlobalAiToolbar` 在替换前对结果的二次 `<think>/<reasoning>` 过滤，避免翻译结果被误删导致展示不完整。
- 2026-03-06 **Doc**: 新增本修复文档，明确“后端清洗、前端直出替换”的职责边界。
