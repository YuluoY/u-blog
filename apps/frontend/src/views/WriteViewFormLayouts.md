# 撰写页表单布局方案

表单在编辑器上方内嵌展示，以下为可选布局方案，在 `WriteView.vue` 中通过 `formLayout` 切换（如 `formLayout = 'card'`）。

---

## 方案 A：卡片分区（默认）

- **结构**：一块白底卡片，内部分为「基础信息」与「发布设置」两区，纵向排列。
- **特点**：层次清晰，适合字段较多的场景。
- **类名**：`write-view__form--card`

```
┌─────────────────────────────────────────────────┐
│ 标题 [________________]  描述 [____________]     │
│ 分类 [▼]  标签 [+选]  封面 [____]  封面预览      │
├─────────────────────────────────────────────────┤
│ 发布时间 [ ] 立即发布  状态 [草稿▼]  私密 置顶   │
│                        [取消] [保存草稿/发布]    │
└─────────────────────────────────────────────────┘
```

---

## 方案 B：单行紧凑

- **结构**：第一行「标题 + 描述（短）」；第二行「分类、标签、状态、时间、私密/置顶」横向排布；第三行操作按钮。
- **特点**：省纵向空间，适合宽屏。
- **类名**：`write-view__form--compact`

```
标题 [_______________]  描述 [________]
分类[▼] 标签[+ ] 状态[▼] 时间[ ] ☑私密 ☑置顶  [保存/发布]
```

---

## 方案 C：两列网格

- **结构**：左列「标题、描述、封面」；右列「分类、标签、发布时间、状态、私密/置顶」；底部按钮横跨两列。
- **特点**：左右分栏，宽屏下信息密度高。
- **类名**：`write-view__form--grid`

```
┌──────────────────────┬──────────────────────┐
│ 标题 [_____________]  │ 分类 [▼]  标签 [+ ]  │
│ 描述 [_____________]  │ 发布时间 [ ] 立即发布 │
│ 封面 [_____] [预览]   │ 状态 [▼]  ☑私密 ☑置顶 │
├──────────────────────┴──────────────────────┤
│              [取消]  [保存草稿/发布]          │
└─────────────────────────────────────────────┘
```

---

## 方案 D：极简 + 折叠

- **结构**：默认只显示「标题 + 状态 + 发布/保存」一行；点击「更多选项」展开下方完整表单（描述、分类、标签、封面、时间、私密/置顶）。
- **特点**：平时极简，需要时再展开。
- **类名**：`write-view__form--minimal`

```
标题 [_________________________]  状态 [草稿▼]  [保存/发布]  [▼ 更多选项]
────────────────────────────────────────────────────────────────────
(展开后) 描述、分类、标签、封面、时间、私密、置顶…
```

---

## 切换方式

- **页面上**：撰写页表单上方有「表单布局」下拉（`u-select`），可选：卡片分区、单行紧凑、两列网格、极简折叠。
- **代码中**：在 `WriteView.vue` 修改 `formLayout` 的初始值，或设置 `showLayoutSwitch = false` 隐藏切换器。

```ts
const formLayout = ref<'card' | 'compact' | 'grid' | 'minimal'>('card')
```

表单容器使用 `write-view__form--${formLayout}`，`WriteSaveForm` 接收 `layout` 并应用 `write-save-form--${layout}` 样式。所有下拉（分类、标签、状态）均使用固定宽度 160px（`.write-save-form__select-fw`），避免选项长度导致布局抖动。
